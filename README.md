Portfolio profesional para QA Engineer construido con **Vue 3**, **Composition API** y **Vite**, aplicando principios de **código limpio** y **arquitectura de componentes**.

## ✨ Características

- 📱 **Responsive** – Diseño adaptable a todos los dispositivos
- 🎨 **Componentes reutilizables** – Arquitectura modular con Single File Components
- 🔄 **Datos centralizados** – Single Source of Truth en archivos de datos
- ♿ **Accesibilidad** – Roles ARIA, navegación por teclado, textos alternativos
- ⚡ **Performance** – Lazy loading, optimización de assets, animaciones eficientes
- 🧩 **Composables** – Lógica reutilizable con Composition API
- 📊 **Métricas QA** – Estadísticas de bugs, casos de prueba, reducción de tiempos

## 🚀 Instalación y uso

```bash
# Clonar el repositorio
git clone https://github.com/marialopez-qa/portfolio-qa-vue.git

# Entrar al directorio
cd portfolio-qa-vue

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Build para producción
npm run build

# Previsualizar build
npm run preview
